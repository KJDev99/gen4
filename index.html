Gemini 3 Flash

Gemini 3 Flash Preview — это быстрый мультимодальный LLM API от Google с низкой задержкой и высокой производительностью, созданный для AI-ассистентов, автоматизации рабочих процессов, программирования и обработки документов без потери качества reasoning-уровня Pro.

Производительность

~218 output tokens/sec

Независимые pre-release тесты показали скорость около 218 output tokens/sec. Это делает Gemini 3 Flash отличным решением для real-time AI workflows, high-QPS backend систем и production-scale задач.

Новые возможности

Улучшения качества

  • Более точные ответы при меньшей задержке. По сравнению с предыдущими версиями Flash, модель даёт более глубокие и качественные ответы при сохранении высокой скорости.
  • Улучшенная работа с документами. Google DeepMind отмечает 15% improvement в accuracy сложных extraction-задач: длинные документы, финансовые отчёты, contracts, handwriting и document parsing.
  • Agent readiness. Модель сохраняет reasoning foundation уровня Pro, оставаясь Flash-fast для multi-step tool usage и AI workflows.

1M-token long context

Подходит для:

  • длинных документов
  • multi-file проектов
  • persistent agent state
  • масштабных document workflows

Multimodal tool/function calling

Поддержка multimodal content внутри function-response flows и различных AI automation сценариев.

Gemini 3 Flash vs Gemini 3 Pro

Gemini 3 Pro — reasoning-first flagship модель. Gemini 3 Flash ориентирован на low latency и coding workflows с фокусом на cost efficiency.

Gemini 3 Flash vs Gemini 2.5 Flash

Gemini 3 Flash — major upgrade предыдущей Flash-линейки. Лучше качество ответов, лучше работа со сложными задачами, сохраняется low cost и high speed.

Gemini 3 Flash vs GPT-5.2

GPT-5.2 выбирают для сложного reasoning и глубокого анализа. Gemini 3 Flash — speed-first модель для задач, где критична responsiveness: скорость, throughput, long context, production workflows.

Ограничения и Guardrails

Gemini 3 Flash использует policy-based safety filtering и может ограничивать генерацию в чувствительных категориях запросов.

При использовании long-context и advanced thinking settings:

  • может увеличиваться latency
  • растёт token usage
  • требуется грамотная fallback-логика для production-приложений

FAQ

Что такое Gemini 3 Flash?
Gemini 3 Flash — быстрый мультимодальный LLM API от Google, оптимизированный для low latency и высокого throughput с контекстом до 1M токенов.
Сколько стоит использование Gemini 3 Flash API?
Input: $0.65 / 1M токенов. Output: $3.90 / 1M токенов. Thinking tokens включены в стандартный pricing.
Какие типы контента поддерживает модель?
Текст, изображения, аудио, видео, PDF-документы. Полная мультимодальность с поддержкой structured outputs и function calling.
Чем Gemini 3 Flash отличается от Pro?
Flash оптимизирован для скорости и cost efficiency. Pro — для глубокого reasoning и максимального качества ответов. Flash подходит для production-scale задач с большим объёмом запросов.
